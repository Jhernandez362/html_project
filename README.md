CostEstimator Landing Page

Landing page responsive desarrollada únicamente con HTML5 y CSS3, enfocada en la estimación visual de proyectos de software.
El proyecto implementa un flujo multi-step completamente funcional utilizando únicamente estados CSS (:checked) sin JavaScript.

🚀 Características del Proyecto
Diseño responsive (Mobile First)
Arquitectura basada en CSS Grid y Flexbox
Navegación suave entre secciones
Wizard multi-step sin JavaScript
Sistema visual de progreso dinámico
Cards interactivas con estados seleccionados
Indicadores visuales de estimación
Layout adaptativo para móvil, tablet y desktop
Landing moderna estilo SaaS
Descarga simulada de reporte PDF
Sección de contacto corporativa
🖥️ Responsive Design

El proyecto fue construido siguiendo una estrategia Mobile First.
